Lenguajes, Frameworks, Bibliotecas y Tecnologias sobre algunas aplicaciones de uso diario

WHATSAPP

Lenguajes

Whatsapp utiliza el lenguaje de Erlang, este es un lenguaje de programación funcional y concurrente, desarrollado por Ericsson en los años 80. Se utiliza para codificar sistemas distribuidos y tolerantes a fallos.

Frameworks

WhatsApp utiliza React Native, un marco de código abierto desarrollado por Facebook, para crear sus aplicaciones móviles. React Native permite a los desarrolladores crear aplicaciones móviles con JavaScript y una base de código que se puede ejecutar en dispositivos Android y iOS.

Bibliotecas

Protocolo extensible de mensajería y presencia (XMPP) WhatsApp utiliza una versión personalizada de este protocolo abierto para intercambiar datos entre los usuarios.

Año de lanzamiento

WhatsApp se lanzó en 2009, inicialmente para iOS, y un año después para Android. La aplicación se popularizó masivamente a partir de 2012.

AMAZON

Lenguajes

Amazon utiliza principalmente los lenguajes de programación Java y Python. Amazon utiliza Python en casi todos los niveles de sus operaciones, como el desarrollo web backend, el código del lado del servidor y el procesamiento de datos. Python es una buena opción para Amazon porque puede manejar grandes volúmenes de datos.

Frameworks

AWS Flow Framework es código abierto, desarrollado y administrado por AWS, y está disponible en Java y Ruby. AWS Flow Framework es un marco de trabajo de código abierto que ayuda a los desarrolladores a crear aplicaciones escalables y tolerantes a errores. Se utiliza en conjunto con Amazon SWF para facilitar la programación asíncrona en el desarrollo de aplicaciones.

Bibliotecas

Para Desarrollo Web y Backend: Spring Framework (Java): Para el desarrollo de aplicaciones web y microservicios. Express.js (Node.js): Framework popular para construir aplicaciones backend en JavaScript. Django / Flask (Python): Utilizados para desarrollo web y servicios API.

Twitter

Lenguajes

Scala: Este es uno de los lenguajes principales en los sistemas backend de Twitter. Fue adoptado por su escalabilidad y por cómo se integra con el ecosistema de Java. Java: Utilizado principalmente para sistemas backend de alto rendimiento y aplicaciones que requieren estabilidad. Python: Se utiliza en varios equipos para tareas de análisis de datos, machine learning, automatización y scripting. JavaScript (Node.js): Utilizado para el desarrollo tanto del lado del servidor (con Node.js) como del lado del cliente (frontend). Ruby: Aunque Twitter comenzó utilizando Ruby on Rails para su desarrollo inicial, con el tiempo se ha alejado de este framework a medida que ha escalado su plataforma. C++ y C: Para componentes de rendimiento crítico que requieren un control a nivel de hardware.

Frameworks

Finagle: Un sistema RPC (Remote Procedure Call) desarrollado internamente por Twitter, escrito en Scala, que ayuda a construir servicios concurrentes y altamente escalables. Apache Storm: Framework de procesamiento de datos en tiempo real que se utiliza para procesar grandes volúmenes de datos en tiempo real, como los tweets. Hadoop: Para el procesamiento y almacenamiento de grandes conjuntos de datos. MySQL: Aunque Twitter ha utilizado bases de datos NoSQL, también se basa en MySQL para datos transaccionales.

Bibliotecas

React: Utilizado para construir la interfaz de usuario de Twitter. React permite a Twitter crear componentes UI reutilizables y manejar eficientemente el estado de la aplicación. Redux: Usado junto con React para manejar el estado global de la aplicación frontend. Node.js: Para el desarrollo de aplicaciones del lado del servidor y para renderizar componentes del lado del cliente de manera eficiente.

TikTok

Lenguajes

Python: Es ampliamente utilizado en TikTok para el desarrollo de algoritmos de machine learning, inteligencia artificial y análisis de datos. Su facilidad de uso y las numerosas bibliotecas disponibles lo hacen ideal para estas tareas. Java y Kotlin: Son los lenguajes principales para el desarrollo de la aplicación de TikTok en dispositivos Android. Kotlin se ha convertido en el estándar moderno para el desarrollo de aplicaciones Android. Swift y Objective-C: Utilizados para el desarrollo de la aplicación de TikTok en iOS. Aunque Swift es el lenguaje preferido, aún se usa Objective-C en algunos casos heredados.

Frameworks

React Native: TikTok utiliza React Native para desarrollar algunas partes de su aplicación móvil, lo que permite la creación de una interfaz de usuario consistente en ambas plataformas, Android e iOS. Flutter: Es posible que algunas partes de la aplicación también estén desarrolladas con Flutter debido a su capacidad para crear interfaces nativas rápidas.

Bibliotecas

Django y Flask (Python): Frameworks utilizados para construir APIs y manejar solicitudes web. Spring Boot (Java): Utilizado para construir microservicios escalables y eficientes. gRPC: Para la comunicación eficiente entre microservicios.

Año de lanzamiento

TikTok se lanzó en septiembre de 2016, pero su versión internacional, con el nombre actual, se lanzó en 2018.
